About me

I’m a 3rd year student of the Video Game Design and Development degree in CITM Barcelona focused on programming of engines like Unity or Unreal. I love programming new things as each one gives me a different challenge than the one before and I enjoy trying to solve it.

Tasks & Responsibilities

Physics Module

  • Helped creation of the physics module of the engine focusing mainly on the components that they had and their interactions with the module. Also worked on the creation of the colliders of the module.

UI Module

  • Created the UI module of the engine with the help of libraries like FreeType too created the text with its respectives components for every single element of the UI, a tweening system and a spritesheet system for animations of 2D elements of the UI. Also in charge of linking the functions from the module to Csharp scripts.

Serialization of the elements of the Physics and UI module

  • Serialized the elements of both modules so that it could be saved and loaded on the scenes.

Sound Module

  • Implemented the sfx and soundtracks of all the levels and entities.

Camera System

  • Created a camera system which could make the use of different cameras on the scene via priority, which means that the camera with the highest priority on the component would automatically transform in the mainCamera and be seen on the viewport.

Bug Fixing and User Experience

  • In the last 2 days of every build I helped around the possible issues that could arise when merging the different branches on the build and in the last two weeks of the project I worked mainly in user experience in order to create the most comfortable experience for players.

Sequencing system (scrapped)

  • I started working for a couple of weeks on an upgrade of the tweening system using sequences so that there could be called multiple tweens from a single command, but it got scrapped due to the lack of time and there were more important features arising at the moment.
